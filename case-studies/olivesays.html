Introduction

Snow Tech built OliveSays to showcase our SaaS development skills and offer entrepreneurs a better way to evaluate their websites. Launched in March 2025 after starting in December 2024, OliveSays uses AI to provide a back-and-forth conversation for website analysis.

It's a practical tool we use internally, and its uniqueness in the AI and website evaluation space made it a fun challenge. This case study covers how we made it happen.

The Vision

We wanted to create a new way to analyze websites, turning it into a conversation rather than a static report. Our mission at Snow Tech focuses on SaaS creation and AI applications, and OliveSays fits that goal.

We targeted entrepreneurs and businesses needing honest, constructive feedback on their sites, delivering a tool that's useful and different.

"The goal was to make website analysis feel like a conversation with an expert, not just a technical report."

The Challenge

Building OliveSays wasn't easy. We faced several hurdles:

  • Data Collection: Websites often block scraping attempts, making data collection tough.
  • Context Management: The AI needed to handle large contexts, which risked overwhelming it with data.
  • User Experience: The interface had to be intuitive for users with no tech background.
  • Financial Viability: We needed to keep costs low and ensure profitability.

Our Solution

We tackled these issues head-on over three months with a clear process:

Prototyping

Started with a basic model to test the concept.

Coding

Used React, JavaScript, and TypeScript for the frontend, hosted on Bluehost. Built web scraping with Puppeteer's stealth plugin, respecting robots.txt to avoid blocks.

AI Integration

Leveraged the OpenAI API instead of training our own model, breaking the chat into phases. Each phase fed the AI relevant data—like aesthetics in one, performance in another—keeping it manageable.

Report Generation

Added a feature to compile insights into downloadable reports.

UI and Structure

Refined the design with HTML and CSS, opting for a dark, non-colorless look that's characterful but not overbearing. Used a rule of thirds layout for the landing page to guide users naturally.

Testing

Tested functionality and load capacity, ensuring it worked smoothly.

To make it user-friendly, we added clear phase indicators and a floating action button for continuation. For profitability, we gave users $5 initial credit and integrated Stripe Customer Portal for a pay-as-you-go tier after that runs out.

Results

OliveSays launched in March 2025, but it's too early for full metrics. We're still reaching out for user feedback and tracking growth. However, it's become a beloved tool internally at Snow Tech and has already been used on a few of our own websites.

The tool handles website evaluations quickly, and its chat feature sets it apart from static analysis tools. It offers real-time, sharp-eyed critiques on aesthetics, performance, SEO, and more, making it a practical asset for Snow Tech and our audience.
